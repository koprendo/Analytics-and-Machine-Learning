# Analytics & Machine Learning Caravanserai

This repository is a hands-on collection of **supervised & unsupervised machine learning models**.  

It also includes analytical matters such as hypothesis testing necessary for ML models. Each project is implemented on real or synthetic datasets, with clear explanations, visualizations, and reusable code structure.

Purpose of this repo:
- Move from **theory to real practice**
- Build a **personal ML portfolio**
- Create a **clean and reproducible structure**
- Learn & experiment with machine learning models
